Instructions
1. Extract all the files in the same folder.
2. Compile the Exer3r2.java file:
	a. Open Command Prompt.
	b. Change directory to the folder where the files are extracted. Type cd (folder path)
	c. Type "javac Exer3r2.java" and press Enter.
3. Run Exer3r2: Type "java Exer3r2" and press Enter.
